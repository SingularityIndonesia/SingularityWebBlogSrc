Saya tidak begitu menyukai Teorema Ketidaklengkapan Godel. Awalnya saya kira hanya karena saya merasa teorema ini tidak membantu sama sekali dalam menemukan solusi untuk mengetahui kebenaran tentang realita -

Namun kini saya menyadari alasan mengapa saya tidak menyukainya. Yaitu karena orang-orang sering salah menggunakan teorema ini dengan mengaitkannya dengan realita.

>Any formal system that is:
>  - consistent (doesn’t prove contradictions),
>  - and powerful enough to express basic arithmetic
>
>will contain true statements that cannot be proven within the system.

Saya tidak menyukainya karena ketika dikaitkan dengan realita -Ia membawa asumsi tersembunyi - Bahwa *struktur* ini (realita) sedari awal bersifat *sistematis*.

Bukan berarti saya menyangkal bahwa alam semesta bersifat sistematis ([[The Pattern Persist]]) - dan bukan berarti juga saya setuju.
